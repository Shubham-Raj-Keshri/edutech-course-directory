# Frontlines Edutech — Course Directory

A modern course discovery experience built with React, Vite, and Tailwind CSS.

This project was created as a frontend-focused EdTech dashboard that emphasizes clean UI patterns, thoughtful spacing, responsive layouts, and polished user interactions. Instead of building a feature-heavy marketplace, the focus was kept on usability, visual hierarchy, and component architecture.

The interface takes inspiration from modern SaaS platforms and developer-focused products, with an emphasis on readability, restrained styling, and smooth interactions.

## Tech Stack

* React + Vite
* Tailwind CSS
* Framer Motion
* Lucide React
* Static JSON data
* Responsive grid layout

---

## Features

* Responsive course directory layout
* Search and filtering
* Category-based filtering
* Instructor filtering
* Sorting by course name and rating
* Pagination
* Skeleton loading states
* Empty and error states
* Sticky navbar and filter section
* Hover interactions and smooth transitions
* Reusable component architecture
* Mobile-friendly layout

---

## Running Locally

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
npm run preview
```

## Project Notes

A few decisions made during development:

* Used static JSON instead of a backend to keep the focus on frontend architecture and UX
* Built reusable UI sections to keep the project scalable
* Added realistic course data and instructor profiles instead of placeholder content
* Kept animations subtle to maintain a production-oriented feel
* Focused heavily on spacing, typography, and responsiveness
